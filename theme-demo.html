Code Example

The theme is controlled by a simple data attribute:

<html data-theme="dark"> or <html data-theme="light">

CSS variables automatically adjust based on theme:

[data-theme="dark"] {
    --bg-primary: #0d1117;
    --text-primary: #c9d1d9;
}

[data-theme="light"] {
    --bg-primary: #ffffff;
    --text-primary: #24292f;
}
